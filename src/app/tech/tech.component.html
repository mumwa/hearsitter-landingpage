Tech

Server

The main server sends requests to series of ML server, balancing requests appropriately. Main server and ml sever connect via gRPC. And ml server made into a docker image, so easy to increase the number of ml servers.

Mobile

The mobile app sends the sound to server, received from the real-time audio stream in seconds. App store recieved result data in local DB using SQLite. App show data and display notifications.

ML

The ml server analyzes this and delivers the results to the main server. For the classification task, we used the EfficientAT model, which uses Knowledge Distillation from Transformers with lightweight CNN for acheiving fast inference time and high performance.
